## Resumen Ejecutivo

**SpaceWork** es un sistema web de gestión de reservas de espacios de trabajo desarrollado para SpaceWork Perú S.A.C., empresa dedicada al alquiler de espacios de coworking en Lima, Perú.

El sistema automatiza completamente el ciclo de reservas: desde la consulta de disponibilidad hasta el pago, evaluación del servicio y notificaciones al cliente, reemplazando procesos manuales basados en Excel, WhatsApp y libretas físicas.

**Tecnologías:** Java SE 8 + Oracle XE + HTML5/Bootstrap 5 + JavaMail (Gmail SMTP)
